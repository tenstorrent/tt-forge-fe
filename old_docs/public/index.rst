Introduction to PyBuda
======================

PyBuda |tm| is a compute framework used to develop, run, and analyze ML workloads on Tenstorrent hardware.

For a quick start, follow :doc:`installation<installation>` steps, and then look at the user guide :doc:`user_guide<user_guide>`.
